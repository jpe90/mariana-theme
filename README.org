* Installation
** Manual installation
1. Download the [[https://raw.githubusercontent.com/jpe90/mariana-theme/master/mariana-theme.el][mariana theme file]]
2. In your =~/.emacs.d/init.el= or =~/.emacs=:

#+begin_src emacs-lisp
  (load-theme 'mariana t)
#+end_src
